# Okiru V2

Okiru V2 é uma versão mais limpa e refinada do Okiru original — um rastreador pessoal de animes, séries, filmes e leituras.

O objetivo foi redesenhar a experiência do zero, mantendo o que funcionava e descartando o que não funcionava, com foco em uma interface mais simples, direta e agradável de usar no dia a dia.

## Sobre

- Rastreamento de animes, séries, filmes e leituras
- Listas personalizadas (assistindo, assistidos, para assistir, lendo, lido, para ler)
- Calendário de lançamentos por episódio
- Perfil com favoritos
- Suporte a tema claro e escuro
- PWA — pode ser instalado como app

## Observações

Este projeto foi feito exclusivamente para uso pessoal. Não há backend, não há contas, tudo fica salvo localmente no seu dispositivo. Nada sai do seu navegador.
